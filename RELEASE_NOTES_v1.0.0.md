Nuzi Inventory v1.0.0

- initial public release of `nuzi-inventory`
- searches bag, bank, and saved alt inventory snapshots
- tracks important items into persistent hotbars
- supports `2 Bars` and `Categories` tracked layouts
- allows configurable icons-per-row wrapping for tracked bars
- equips tracked gear directly from the current character's live bag
- keeps tracked bar positions and settings persistent in `.data`
- automatically migrates legacy `nuzi-bagsearch` settings into `nuzi-inventory`
